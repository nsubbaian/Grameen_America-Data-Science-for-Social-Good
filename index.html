The Introduction

Grameen America is a 501 nonprofit microfinance organization based in New York City.
It was founded by Nobel Peace Prize recipient Muhammad Yunus in 2008.
Grameen America is run by former Avon Chairman and CEO Andrea Jung.

This website provides an overview and analysis of the data collected by Grameen America.
